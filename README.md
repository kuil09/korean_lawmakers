# 왜 이런 것을 남기는가? 이것은 무엇인가
유래 없는 일이 벌어졌습니다. 개발자로서 사회에 기여할 수 있는 일이 무엇일까 생각해 봤습니다. 
역사에 정쟁이 있었다라는 한마디로 끝내지 않고자 합니다. 탄핵안 표결에 불참했던 이들의 의견이 밝혀지고 손바닥 뒤집듯 거취를 수정할 때 마다 commit 이력으로 남깁니다.

# 기여하는 법
위의 정치인의 발언(뉴스, SNS)의 내용이 밝혀지면 rebase 후 readme를 수정해서 PR을 보내주시면 됩니다.
commit은 의원 개인의 발언 단위로 해주셔야만 merge가 됩니다. 기타 수정이 필요한 부분은 issue를 작성해 주세요.
